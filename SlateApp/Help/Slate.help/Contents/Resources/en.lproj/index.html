Quick start

  1. Open the model menu at the bottom of the sidebar.
  2. Choose an installed GGUF model, import a local file, or open Downloads.
  3. Create a Chat or Code conversation and type a message.

Slate loads one language model at a time to keep memory use predictable. The first load can take a moment; later turns reuse the loaded model.

Chat, Code and files

Chat

Use Chat for questions, writing, summaries and conversations with local files. Drag supported files into the composer or add them with the + button.

Code

Code conversations can inspect a folder you select, propose edits and run local commands. Slate shows every tool action and asks before consequential work unless you explicitly change the permission mode.

Treat model output as untrusted. Review commands and file changes before approving them, especially in folders containing important data.

Models and downloads

Slate does not include model weights. You can import compatible local GGUF files or explicitly download a curated model. Before a curated download, Slate shows the provider model card and licence. Downloaded files remain governed by those provider terms.

  • Installed shows local models and RAM fit.
  • Browse requires Model & voice downloads to be enabled.
  • Image installs the complete compatible image bundle.

Deleting a model moves it to the Trash. It does not delete conversations.

Flow dictation and voice

Flow records from your microphone and transcribes on-device. Optional Parakeet speech recognition, Supertonic voices and Silero VAD are downloaded only when you request them. Their provider licence is shown in Slate.

Set Speech language to Auto for mixed use, or choose English/German explicitly when a small model keeps replying in the wrong language. The language choice guides transcription; the language of the answer still depends on the prompt and model.

Microphone, Accessibility and Input Monitoring permissions are managed by macOS in System Settings → Privacy & Security.

Network Access and Silent Mode

Settings → Network Access is the central network control.

  • Silent Mode blocks Slate's outbound update, licence, model/voice-download and cloud-connector traffic.
  • Local chat, Code, images with installed models, Flow with installed speech files, search and local tools continue to work.
  • Links you open leave Slate and follow the network rules of the destination app.

Silent Mode does not turn macOS into a firewall and cannot govern arbitrary scripts you approve in Code. It prevents Slate's own built-in network clients and cloud connectors from starting requests.

Appearance and Slate Quick

In Settings → General, choose Light, Dark or System and create a custom palette for canvas, panels, accent, your messages, Slate responses and tool activity. Slate automatically selects readable foreground text for each colour.

Press ⌥ Space for Slate Quick from any Space, including full-screen apps. Click outside the panel or press Esc to close it. Quick always uses the in-process local model and never falls back to a cloud connector.

Useful shortcuts

  • ⌘ N new conversation
  • ⌘ , Settings
  • ⌥ Space Slate Quick
  • Esc close transient panels
  • Return send; use the composer control for a line break when configured

Privacy and data

Conversations, memories, audit records, settings and imported knowledge stay on this Mac. API keys and licence records use the macOS Keychain. Export, import and delete controls are in Settings → Privacy & Data. Cloud mode is separate, off by default, and may send the prompt and selected context to the connector you choose.

Troubleshooting

A model will not load

Check the RAM fit badge, try a smaller quantization, and close memory-heavy apps. Slate can automatically unload the model under critical memory pressure.

Voice uses the wrong language

Choose an explicit speech language in Dictation settings. Then tell the chat model the desired response language in the prompt or system instruction.

Downloads or updates do nothing

Open Network Access and turn off Silent Mode. Enable Model & voice downloads for optional files.

macOS asks for a password after every development build

Rebuilding with a different ad-hoc signature changes Slate's identity and macOS may ask again for protected permissions. A stable local code-signing identity preserves them; the public Developer ID build will also have a stable identity.

Report a bug

Use Settings → Bug Reports. Nothing is sent automatically. You can also write to info@lange-co-consulting.de.
